## ğŸ› ï¸ Installation & Setup

### Prerequisites
- **Node.js** (v16 or higher)
- **Python** (v3.8 or higher)
- **MongoDB** (local or cloud instance)
- **Git** for version control

### 1. Clone the Repository
```bash
git clone https://github.com/sanjaydaud7/Brainbox-.git
cd brainbox
```

### 2. Backend Setup
```bash
# Navigate to backend directory
cd Backend

# Install dependencies
npm install

# Create environment file (.env)
# Add your MongoDB connection string, JWT secret, etc.

# Start the backend server
npm start
# or for development
npm run dev
```

### 3. ML Service Setup
```bash
# Navigate to MoodModel directory
cd MoodModel

# Install Python dependencies
pip install tensorflow opencv-python numpy

# Start ML service
python main.py
```

### 4. Frontend Setup
```bash
# Navigate to frontend directory
cd Frontend

# Open index.html in a web browser or use Live Server extension in VS Code
# No additional setup required for vanilla HTML/CSS/JS
```

## ğŸ¨ Features in Detail

### Mood Detection System
The mood detection system uses advanced computer vision and machine learning:

1. **Image Capture**: Uses device camera to capture user's image
2. **Face Detection**: Identifies face region using Haar cascade classifier
3. **Feature Extraction**: Processes facial landmarks and expressions
4. **Emotion Classification**: Predicts emotions using trained CNN model (30 epochs)
5. **Real-time Analysis**: Provides instant mood feedback
6. **Historical Tracking**: Stores and analyzes mood patterns over time

### AI Chatbot Integration
Intelligent mental health support chatbot:

1. **Natural Language Processing**: Understands user queries
2. **Context Awareness**: Maintains conversation history
3. **Mood-Responsive**: Adapts responses based on detected mood
4. **Crisis Detection**: Identifies and responds to crisis situations
5. **Resource Recommendations**: Suggests relevant resources

### Admin Dashboard
Comprehensive administrative interface:

1. **User Management**: View and manage registered users
2. **Appointment Oversight**: Monitor and manage appointments
3. **System Analytics**: Track platform usage and engagement
4. **Content Management**: Update resources and content
5. **Security Controls**: Admin authentication and authorization

### Mental Health Reports

1. **Mood Analysis**: Historical mood pattern analysis
2. **Progress Tracking**: Monitor improvement over time
